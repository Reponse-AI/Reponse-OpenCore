## Vue d'ensemble

Reponse peut transférer des événements commerce vers Klaviyo pour alimenter vos flux marketing (panier abandonné, post-achat, réactivation) depuis votre compte Klaviyo existant.

## Configuration

Ajoutez votre clé API Klaviyo dans les réglages d'intégrations du dashboard. Les événements sont transférés automatiquement une fois connecté.
