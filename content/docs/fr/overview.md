## Vue d'ensemble

Reponse est une plateforme Headless Commerce + AI. L'API donne un accès programmatique au catalogue, aux paniers, au checkout, aux commandes, à la fidélité, au support et à la couche d'activation AI, pour construire storefronts, agents et intégrations au-dessus de votre boutique.

## URL de base

Toutes les requêtes API vont vers `https://api.reponse.ai`. Certaines opérations liées à la boutique fonctionnent aussi via `https://votre-boutique.reponse.ai`.

## Authentification

Authentifiez chaque requête avec une clé API workspace en token Bearer. Voir le guide Authentification pour les types de clés et les scopes.

## Par où commencer

Nouveau ? Suivez Installation, puis Votre première requête. Explorez ensuite le SDK et la référence API de chaque ressource.
