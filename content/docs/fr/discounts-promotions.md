## Vue d'ensemble

Reponse supporte à la fois les codes de réduction historiques et un moteur de promotions plus récent. Les codes peuvent être saisis par le client ou automatiques. À l'application d'un code sur un panier, Reponse vérifie d'abord la table promotions, puis se rabat sur les codes de réduction.

## Automatique vs code

Les remises automatiques s'appliquent sans saisie quand leurs conditions sont remplies. Les remises à code nécessitent que l'acheteur saisisse un code au checkout.

## Application sur un panier

Utilisez l'endpoint promotions du panier pour valider et appliquer un code. Les remises automatiques sont reflétées automatiquement à la lecture d'un panier.
