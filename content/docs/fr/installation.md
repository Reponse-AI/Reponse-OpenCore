## Vue d'ensemble

Installez le SDK avec votre gestionnaire de paquets préféré et fournissez votre clé API via une variable d'environnement.

## Installation

```bash
npm install @reponseai/sdk
# ou : pnpm add @reponseai/sdk
# ou : yarn add @reponseai/sdk
```

## Configuration

Stockez votre clé dans `REPONSE_API_KEY` et lisez-la à la construction du client. Pour les apps React, utilisez `@reponseai/react`.
