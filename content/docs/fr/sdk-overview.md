## Vue d'ensemble

Reponse fournit un SDK TypeScript (`@reponseai/sdk`), des hooks React (`@reponseai/react`) et un serveur MCP. Le SDK est généré depuis la spécification OpenAPI, donc ses types correspondent toujours à l'API.

## Choisir un outil

Utilisez le SDK TypeScript pour le code serveur et full-stack, les hooks React pour le data fetching UI, et le serveur MCP pour connecter des agents IA.
