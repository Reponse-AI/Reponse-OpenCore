## Vue d'ensemble

Le widget de chat apporte la couche d'activation Reponse sur n'importe quelle page. Il répond aux questions des acheteurs en contexte et peut les guider vers l'achat. Intégrez-le avec un petit script.

## Intégration

Ajoutez le snippet d'intégration depuis votre dashboard à votre site. Le widget hérite automatiquement du thème de votre workspace.
