## Vue d'ensemble

Reponse fournit un serveur Model Context Protocol (MCP) qui expose les opérations commerce aux agents et IDE compatibles MCP (comme Claude ou Cursor). Il permet à un agent de lire le catalogue et d'agir sur les paniers et commandes via une interface d'outils typée.

## Connexion

Pointez votre client compatible MCP vers le serveur MCP de Reponse et authentifiez-vous avec votre clé API workspace. Les outils correspondent aux mêmes opérations `/v1` documentées ici.
